Aplikasi ini menggunakan teknologi YOLO untuk mendeteksi objek secara real-time dari kamera. Sistem akan mengumumkan objek yang terdeteksi melalui suara dan memberikan peringatan jika ada objek terlalu dekat.

ğŸš€ Cara Memulai

  1. Izinkan Akses Kamera - Saat pertama kali membuka aplikasi, browser akan meminta izin untuk mengakses kamera. Klik "Allow" atau "Izinkan".
  2. Tunggu Model Dimuat - Aplikasi akan otomatis memuat model YOLO. Tunggu hingga status "Model" menunjukkan "Siap" (hijau).
  3. Kamera Siap - Pastikan status "Kamera" menunjukkan "Webcam Aktif" atau "ESP32-CAM Aktif".
  4. Mulai Deteksi - Klik tombol "Live Detection" untuk mulai mendeteksi objek secara real-time.

ğŸ“¢ Pengumuman Objek (100-150cm)

Sistem akan mengumumkan nama objek dalam Bahasa Indonesia jika objek berada dalam jarak 100-150cm. Contoh: "manusia", "kursi", "mobil".

âš ï¸ Peringatan Tabrakan (<50cm)

ğŸ“ Estimasi Jarak

Sistem menggunakan triangle similarity untuk memperkirakan jarak ke objek:

  • Jarak ditampilkan pada setiap bounding box
  • Akurasi tergantung pada kalibrasi focal length
  • Untuk akurasi lebih baik, lakukan kalibrasi dengan objek yang jaraknya diketahui

ğŸ’¡ Kalibrasi Jarak: Buka browser console (F12) dan jalankan:

calibrateFromKnownDistance(jarak_cm, class_id, bbox_width_px, bbox_height_px)

Contoh: calibrateFromKnownDistance(100, 0, 150, 400)

ğŸ¯ Tips Penggunaan

  • Pencahayaan: Pastikan pencahayaan cukup untuk deteksi yang lebih baik
  • Jarak Objek: Objek terlalu dekat atau terlalu jauh mungkin tidak terdeteksi dengan baik
  • Posisi Kamera: Pastikan kamera stabil dan tidak goyang
  • Performa: Tutup tab/aplikasi lain untuk performa lebih baik
  • ESP32-CAM: Gunakan mode Stream untuk performa lebih baik, atau mode Capture jika stream tidak stabil
